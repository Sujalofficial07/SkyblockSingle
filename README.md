# SkyblockSingle

An offline, singleplayer Hypixel SkyBlock-inspired remake for Minecraft 1.20.1 (Fabric).

## Features
- Void World Generation
- Starter Island
- Coins System
- Persistent Data
- Skills (Coming soon)
- Minions (Coming soon)

## Commands
- `/skyblock` - Mod info
- `/island reset` - Reset your island and data
- `/coins` - Check your balance
- `/coins add <amount>` - Add coins (Debug)

## Build
Run `./gradlew build` to generate the mod jar in `build/libs/`.
